# Tecnetium - Website Generator

#### Video Demo:  https://youtu.be/X2X9owtZ-gs

Tecnetium is a tool to permit generate a showcase website for little companies just by entering their informations and letting the program create their website. Follow the video demo to use it. I realised it for my HarvardX Professional Certificate in "Computer Science for Artificial Intelligence".

## Install & Use
```bash
git clone https://github.com/arcabaffe/tecnetium.git

cd tecnetium 

pip3 install -r requirements.txt

python3 main.py
```

go to http://localhost:5000
